Requirements:

        - C++ compiler:

             - Windows:

                  Compiled with Visual C++ 2005 Express Edition

                  (http://msdn.microsoft.com/vstudio/express/default.aspx).

             - Mac OS X:

                  Compiled with XCode 2.4.1

                  (http://www.apple.com/es/macosx/features/xcode/).

             - Other:

                  It uses Boost, which is a multiplatform library (refer to

                  Boost's website), and the Standard Template Library.



	- Boost C++ Libraries: http://www.boost.org/

             - Windows:

                  Included Visual C++ Express project searches for binary libs

                  in '%BOOST_BASE_DIR%\lib' and headers in

                  '%BOOST_BASE_DIR%\include'.

             - Mac OS X:

                  Included XCode project searches for binary libs in

                  '/opt/local/lib' and '/usr/local/lib',

                  and headers in

                  '/opt/local/include' and '/usr/local/include'.

                  I recommend you to install Darwin Ports version of Boost.
